Welcome to StegoSafe 🚀

Securely hide and retrieve messages with cutting-edge technology.

Step 1: Choose Image 📁

Click on the "Choose Image" button or drag and drop an image. Ensure it is in .png or .jpg format.

Step 2: Enter Message ✏️

Type the message you want to hide in the text box. Ensure it fits within the image's storage capacity.

Step 3: Add Security Key 🔑

For added protection, enter a security key. This will be required during decryption.

Step 4: Encrypt & Download 💾

Click the "Encrypt and Download" button. Your encrypted image will be downloaded securely.

Step 5: Upload Encrypted Image 📤

To retrieve the hidden message, upload the encrypted image by clicking "Choose Image".

Step 6: Enter Security Key 🛡️

If a security key was used during encryption, enter it in the key field. Leave blank if no key was used.

Step 7: Decrypt Message 🔓 & View Message 👀

Click the "Decrypt" button. The system will process the image and retrieve the hidden message.

Step 8: Validate and View Message 🔐

The hidden message will be displayed in the text box. If incorrect, an error message will be shown.
