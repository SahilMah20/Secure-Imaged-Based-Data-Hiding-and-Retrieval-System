Secure Image-Based Data Hiding and Retrieval System
Overview

Secure Image-Based Data Hiding and Retrieval System is a web application developed as a TY B.Sc. Information Technology academic project under the University of Mumbai.

The application securely hides confidential text inside digital images using image steganography combined with Data Encryption Standard (DES) encryption.
Only users possessing the correct security key can retrieve the hidden information, ensuring confidentiality and protection against unauthorized access.

Features
Secure text encryption using DES
Image steganography for hiding confidential messages
Password-based retrieval
Encryption and decryption modules
User-friendly web interface
Fast processing
